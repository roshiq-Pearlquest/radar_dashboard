## 🚀 Getting Started

### Prerequisites

- Python (v3.7 or higher) and pip

### Installation

```bash
# Clone the repository
git clone https://github.com/USERNAME/radar_dashboard.git
cd radar_dashboard

# Install dependencies
pip install -r requirements.txt
```

### Running

```bash
python main.py
```
